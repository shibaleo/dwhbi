# リスク管理表

## 方針

プロジェクト遂行上のリスクを識別し、発生確率と影響度でスコアリングする。高リスク項目には事前対策（軽減策）と発生時対応（緊急対応策）を定義する。

## リスク分類

| 分類 | 内容 | 管理場所 |
|------|------|----------|
| プロジェクトリスク | スケジュール遅延、リソース不足、要件変更 | 本ドキュメント |
| 運用リスク | サービス終了、無料枠超過、可用性 | [430 外部サービス依存リスク](/01-product/400-operations/430-risk/service-dependency) |
| セキュリティリスク | 認証漏洩、不正アクセス | [500 脅威モデリング](/01-product/500-security/threat-model) |
