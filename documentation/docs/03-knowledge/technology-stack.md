# 技術スタック

本プロジェクトで使用する技術の一覧と概要。

## 言語

| 技術 | 概要 |
|------|------|
| **TypeScript** | JavaScript に静的型付けを追加した言語 |
| **Python 3.12** | データ処理・機械学習に強い汎用スクリプト言語 |
| **SQL** | リレーショナルデータベースの標準クエリ言語 |
| **Typst** | LaTeX の代替となる現代的な組版システム |

## Web フレームワーク

| 技術 | 概要 |
|------|------|
| **Next.js 15** | React ベースのフルスタック Web フレームワーク |
| **Hono** | 軽量・高速な Edge 対応 Web フレームワーク |
| **Supabase** | PostgreSQL ベースの BaaS（認証・DB・ストレージ） |

## 機械学習・データ処理

| 技術 | 概要 |
|------|------|
| **LightGBM** | 勾配ブースティング決定木の高速実装 |
| **scikit-learn** | Python の標準的な機械学習ライブラリ |
| **pandas** | Python のデータ操作・分析ライブラリ |
| **dbt** | SQL ベースのデータ変換ワークフローツール |

## インフラ

| 技術 | 概要 |
|------|------|
| **Oracle Cloud VM** | Oracle 提供の Always Free 枠を持つ IaaS |
| **Cloudflare Tunnel** | ポート開放不要でローカルサーバーを公開するトンネル |
| **Vercel** | Next.js に最適化されたサーバーレスホスティング |
| **Grafana Cloud** | メトリクス・ログ・トレースの統合監視 SaaS |

## CI/CD・開発ツール

| 技術 | 概要 |
|------|------|
| **GitHub Actions** | GitHub 統合の CI/CD ワークフロー |
| **Nx** | モノレポ向けビルドシステム・タスクオーケストレーター |
| **Docker Compose** | 複数コンテナの定義・実行ツール |

## テスト

| 技術 | 概要 |
|------|------|
| **Vitest** | Vite ベースの高速 JavaScript テストフレームワーク |
| **pytest** | Python の標準的なテストフレームワーク |

## リンター・フォーマッター

| 技術 | 概要 |
|------|------|
| **ESLint** | JavaScript/TypeScript の静的解析ツール |
| **Ruff** | Rust 製の超高速 Python リンター・フォーマッター |

## 開発環境

| 技術 | 概要 |
|------|------|
| **VSCode Remote SSH** | リモートサーバー上でローカル同様に開発できる拡張機能 |
