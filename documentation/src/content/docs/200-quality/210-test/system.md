# 総合テスト計画

## 方針

本番相当の環境で、システム全体が要件を満たすことを検証する。全サービスの同期 → データ変換 → 分析 → 可視化までのエンドツーエンドフローを通しで実行し、期待されるデータが正しく生成されることを確認する。リリース前のチェックリストとして使用し、手動で実施する。

## 検証項目

| 項目 | 検証内容 |
|------|----------|
| 同期完了 | 8サービスすべてが正常に同期完了 |
| データ変換 | staging/core 層のデータが正しく生成 |
| 分析出力 | estimate が正しく計算される |
| UI表示 | console で同期状況・設定が正しく表示 |
